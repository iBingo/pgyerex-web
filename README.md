# Privacy Gen

一个基于 Vue 3、Vite 和 TDesign 的移动应用管理平台，涵盖法务文档生成、国际化管理、图标资源管理、应用分发与特性开关等能力。

本项目由小米 MiMo 和 GPT 辅助生成。

## 功能模块

### 法务文档

- 隐私政策生成器 — 填写应用信息与第三方 SDK，实时预览并导出
- 用户协议生成器 — 填写应用与服务信息，实时生成协议文本
- 支持复制到剪贴板，导出为 `Markdown`、`TXT`、`HTML`
- 支持按项目管理多份文档

### 国际化管理

- 多语言文案集中管理与状态追踪
- 导入词条文件，校验缺失项、冲突项和语言覆盖率
- 任务管理（导入、校验、发布、导出）
- 按语言、环境和版本导出国际化资源包

### 图标资源

- 按项目隔离管理图标资源库
- 支持导入 Figma 文件
- 导出 Compose ImageVector 或 Android Vector XML

### 应用图标

- 上传 1024px 源图，一键生成 iOS、Android、Web App、Cordova 等平台图标
- 可配置圆角、内边距、背景等生成参数

### 应用启动页

- 管理启动图资源与版本支持
- 可视化编辑背景、图层位置、缩放和设备尺寸

### 移动应用中心

- 应用列表与详情管理（基本信息、部署环境、版本记录）
- 版本管理与安装包发布
- 安装页管理（链接、截图、下载状态）
- AB 测试（实验流量、版本策略、分组命中、转化指标）

### 功能特性管理

- 特性开关管理（环境规则、Variation、实验层）
- 人群组与人群属性管理
- 开关审批流程（发布、灰度扩大、规则变更）

## 技术栈

- Vue 3
- Vite 5
- Vue Router 4
- TDesign Vue Next
- TDesign Mobile Vue
- tdesign-icons-vue-next

## 安装与运行

```bash
# 安装依赖
npm install

# 启动开发环境
npm run dev

# 构建生产版本
npm run build

# 本地预览构建结果
npm run preview
```

## 说明

本项目适合用作移动应用管理的前端原型或内部工具。法务文档为模板化生成，不构成正式法律意见，若用于正式业务发布建议由法务或合规人员审阅修订。
